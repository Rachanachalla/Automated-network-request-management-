# IDEATION PHASE

## Project Title
Automated Network Request Management

## Problem Statement
Organizations receive many network-related requests such as VPN access, firewall changes, IP allocation, and router configuration. Handling these requests manually consumes time, creates delays, and increases the possibility of human errors.

## Proposed Solution
Develop a ServiceNow-based automated request management system that allows users to submit network requests through a catalog item. The system validates the request, routes it automatically for approval, assigns tasks to the appropriate team, and notifies users about request status.

## Objectives

- Reduce manual work
- Improve approval efficiency
- Standardize request processing
- Provide real-time request tracking
- Improve customer satisfaction

## Technologies Used

- ServiceNow
- Flow Designer
- Service Catalog
- Client Scripts
- UI Policies
- Notifications
